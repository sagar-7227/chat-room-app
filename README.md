# Simple Chat-Room App
This is a simple chat-Room app built with React and Firebase.

## Requirements
1. React
2. Firebase
3. MongoDB

## Installation
1. Clone the repository
```
git clone https://github.com/sagar-7227/chat-room-app.git
```
## Quick Overview

You should see something like this now:

npm start
